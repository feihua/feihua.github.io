# 本地开发

## 下载代码
```shell
git clone https://github.com/feihua/flutter_mall.git
```

## 修改配置
**修改后端接口地址：flutter_mall\lib\config\service_url.dart**

![image-20240102112230352](dev.assets/image-20240102112230352.png)

## 启动项目
```shell
flutter pub get
flutter run
```
