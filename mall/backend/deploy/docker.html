# 在docker上部署

# 安装docker

curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
1

# 构建镜像

在linux下提供makefile脚本,构建docker镜像

make image
1
2

# 启动服务

在linux下提供makefile脚本,启动docker镜像

make run
1
